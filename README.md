# X 纵向长图切割工具 (Slice-X)

一个简单易用的网页工具，用于将长图切割成适合 X (Twitter) 推文展示的多张切片。

## 功能特点

- **自动分割**：根据设定的份数（2-4份）自动计算切割高度。
- **自定义设置**：支持调整切割份数和图片间的模拟间隙。
- **实时预览**：拖拽图片后即可查看切割效果。
- **一键下载**：生成的切片可以一键全部下载。
- **纯前端处理**：所有图片处理都在浏览器本地进行，不上传服务器，保护隐私。

## 使用方法

1. 打开网页。
2. 将长图拖入上传区域，或点击上传。
3. 调整切割份数和间隙大小。
4. 满意后点击“一键下载全部切片”。

## 技术栈

- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (Canvas API)
