# X 纵向长图切割工具 (Slice-X)

一个简单易用的网页工具，用于将长图切割成适合 X (Twitter) 推文展示的多张切片，或将多张切片拼接还原成长图。

## 功能特点

### 切割模式
- **自动分割**：根据设定的份数（2-4份）自动计算切割高度。
- **自定义设置**：支持调整切割份数和图片间的模拟间隙。
- **实时预览**：拖拽图片后即可查看切割效果。
- **一键下载**：生成的切片可以一键全部下载。

### 拼接模式 🆕
- **模板选择**：支持纵向和横向两种拼接方向。
- **横向拼接**：将多张图片横向连接成一张宽图。
- **纵向拼接**：将多张图片纵向连接成一张长图。
- **可拖拽排序**：支持拖拽调整图片顺序。
- **间隙填充**：支持在图片间添加填充间隙。

### 通用特性
- **纯前端处理**：所有图片处理都在浏览器本地进行，不上传服务器，保护隐私。
- **多语言支持**：支持中文和英文界面切换。

## 使用方法

### 切割模式
1. 打开网页。
2. 将长图拖入上传区域，或点击上传。
3. 调整切割份数和间隙大小。
4. 满意后点击"一键下载全部切片"。

### 拼接模式
1. 切换到"拼接模式"。
2. 选择拼接方向（纵向/横向）。
3. 拖入或选择多张图片。
4. 调整图片顺序和间隙大小。
5. 点击"下载还原长图"。
